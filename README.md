# dotfiles

Managed via [chezmoi](https://www.chezmoi.io/).
Includes automatic setup for Arch and Alpine.

## Installation

Run this one-liner on a fresh machine:

```bash
sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply BartiX259
```

## Features

*   **WM:** Niri
*   **Editor:** Neovim (External repo)
*   **Terminal:** Alacritty
*   **Shell:** Fish
*   **Bar:** Waybar
*   **OSD:** Eww
*   **File explorer:** Nemo
*   **Browser:** Firefox
```
